🧠 O que é uma Skill de IA?
Entenda o conceito antes de criar a sua
📌

O que é?

Uma skill é um arquivo SKILL.md que ensina o Claude a executar uma tarefa específica do seu modo. Ela é ativada automaticamente quando você faz um pedido compatível.

⚙️

Como funciona?

O Claude lê a description da skill e decide se deve ativá-la. Se sim, segue todas as instruções contidas no arquivo.

📂

Estrutura

Uma pasta com SKILL.md (obrigatório), mais assets/ para templates e logos, e references/ para documentos extras.

💡

Exemplo Real

A skill instrucoes-douglas-vilar é ativada sempre que você pede um contrato, petição ou parecer — aplicando papel timbrado e formatação automaticamente.

--- frontmatter YAML name: minha-skill # identificador único description: # MAIS IMPORTANTE O que faz. Usar SEMPRE para contratos, petições... --- # Título da Skill ## Visão Geral ## Regras Fundamentais ## Comandos Rápidos
💡
Dica do Douglas Vilar: A description é o campo mais crítico. Use "SEMPRE" para torná-la obrigatória em determinados contextos. Seja específico — o Claude tende a subutilizar skills com descrições vagas.
